## 📌 사용자 스토리 & 인수 조건

> 요구사항 1: 수입/지출 입력 기반 자산 시뮬레이션 기능

사용자 스토리:

“나는 사회 초년생으로서, 미래에 어떤 집을 살 수 있을지 예측하기 위해 수입과 지출을 입력하고 자산 시뮬레이션 기능을 원한다.”

인수 조건:

- Given: 사용자가 ‘내 집 마련 시뮬레이터’에 로그인해 수입과 지출 정보를 입력하고 있다.
- When: 사용자가 월급, 고정지출, 저축 금액 등을 입력하고 ‘시뮬레이션 실행’ 버튼을 누른다.
- Then: 앱은 자산 증가 그래프와 예상 구매 가능 주택 정보(예: 위치, 가격대)를 시각화하여 보여준다.

---

> 요구사항 2: 청약 가점 자동 계산 기능

사용자 스토리:

“나는 청약 준비자로서, 내 청약 가점을 쉽게 알고 싶기 때문에 자동 계산 기능을 원한다.”

인수 조건:

- Given: 사용자가 무주택 기간, 부양가족 수, 청약 통장 가입 기간 등을 입력하고 있다.
- When: 사용자가 ‘가점 계산’ 버튼을 클릭한다.
- Then: 시스템은 항목별 점수를 자동으로 계산하고 총 청약 가점을 출력한다.

---

> 요구사항 3: 정부 대출 조건 적용 시뮬레이션

사용자 스토리:

“나는 신혼부부로서, 디딤돌 대출이 적용되는 조건에서 얼마나 대출받을 수 있는지 알고 싶다.”

인수 조건:

- Given: 사용자가 소득, 세대 구성, 주택 가격 정보를 입력하고 있다.
- When: 사용자가 ‘정부 대출 시뮬레이션’ 항목을 선택한다.
- Then: 앱은 적용 가능한 대출 상품 목록과 예상 대출 한도, 월 상환액을 계산하여 보여준다.

---

> 요구사항 4: 학군 기반 지역 추천 기능

사용자 스토리:

“나는 무주택 중년층으로서, 자녀 교육에 좋은 지역을 알고 싶기 때문에 학군 기반의 지역 추천 기능을 원한다.”

인수 조건:

- Given: 사용자가 자녀 나이, 현재 학교, 희망 학군 정보를 입력하고 있다.
- When: 사용자가 ‘학군 추천’ 기능을 클릭한다.
- Then: 시스템은 주요 학군 및 해당 지역의 평균 주택 가격과 매물 정보를 추천 리스트로 보여준다.

## ✅ 핵심 가치 제안

1. **현실 기반 계획 지원**: 사용자 수입, 지출, 자산 데이터를 기반으로 실제 구매 가능성을 예측해 실현 가능한 내 집 마련 계획을 제시
2. **맞춤형 청약/대출 시뮬레이션**: 사용자의 조건에 맞춰 복잡한 청약 제도와 대출 가능 여부를 자동으로 분석하고 추천
3. **미래 전망 시각화**: 시간에 따른 자산 성장과 주택 구매 가능 시점/지역을 직관적인 그래프로 제공하여 목표를 명확하게 설정

## ✅ 핵심 기능

1. **자산 성장 시뮬레이션**: 수입/지출 데이터를 기반으로 수년간의 자산 변화와 주택 구매 가능 시점을 그래프로 시각화
2. **청약 가점 및 조건 자동 계산기**: 무주택 기간, 부양가족 수 등 입력하면 가점을 자동 계산하고, 조건에 맞는 청약 단지를 추천
3. **정부 대출 상품 적용 시뮬레이션**: 디딤돌·보금자리론 등 대출 가능 여부를 판단하고, 대출 한도와 월 상환액을 구체적으로 예측

## ✅ 구현 범위

- 내 집 마련의 UI/UX와 기본적인 유저 플로우를 보여주는 프로토타입 수준
- 사용자 입력에 대한 미리 정의된 응답 패턴 활용
